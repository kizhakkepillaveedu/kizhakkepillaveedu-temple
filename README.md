# Annapurna Temple Project

This repository contains the code for the Family Temple project, including:
- `family-temple`: Static frontend assets.
- `family-temple-app`: Next.js web application.

## Getting Started

### Static Frontend
To view the static frontend, you can use a local server:
```bash
npm run dev
```

### Next.js App
To run the web app:
```bash
cd family-temple-app
npm install
npm run dev
```
